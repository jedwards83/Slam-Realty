SLAM REALTY PROJECT
-------------------

The Purpose of this project is to be an easy access method to input your contact information and your interest in either Buying, Selling, Renting a house. There is also an option for investors who want to support the company.

Each Image has a button embedded into it that will populate a form. That form is there to transmit data back through php to a database(This will be implemented after Code Louisville Completion). Currently We just have the functionality of the form without the storing of any data.

Each Button in the top right will link to a page that will have the form from the main index page unhidden with Lorem Ipsum on each page which would outline why people would Buy, Rent, Sell, or Invest in Slam Realty.
